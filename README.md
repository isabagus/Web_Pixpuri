# PixPuri Website

Struktur file dan dokumentasi singkat untuk website PixPuri.

## Struktur File

- `index.html` — Struktur utama HTML website.
- `style.css` — Seluruh styling (CSS) website.
- `script.js` — Seluruh interaksi dan logika JavaScript website.

## Penjelasan File

### index.html
- Hanya berisi struktur HTML utama.
- Semua style diambil dari `style.css`.
- Semua script diambil dari `script.js`.
- Tidak ada kode CSS/JS inline.

### style.css
- Berisi seluruh styling website.
- Menggunakan variabel CSS untuk warna, radius, dsb.
- Komentar diberikan pada setiap section utama (Navbar, Hero, About, Catalog, Contact, Footer, Responsive, dsb).

### script.js
- Berisi seluruh kode JavaScript untuk interaksi website.
- Komentar diberikan pada setiap blok fungsi utama (navbar scroll, toggle menu, filter produk, animasi reveal, form kontak, dsb).

## Cara Menjalankan

1. Buka file `index.html` di browser.
2. Pastikan file `style.css` dan `script.js` berada di folder yang sama dengan `index.html`.

---

Website ini statis, tidak membutuhkan backend/server khusus.
